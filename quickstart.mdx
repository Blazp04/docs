# Quick Start Guide

Get your SkyGuard development environment up and running quickly with this step-by-step guide.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** (version 18 or higher)
- **npm** or **yarn** package manager
- **Docker** and **Docker Compose**
- **Git** for version control
- **Visual Studio Code** (recommended) or your preferred code editor

## Development Setup

### Step 1: Clone the Repository

<AccordionGroup>
  <Accordion icon="github" title="Clone SkyGuard Backend">
    ```bash
    git clone https://github.com/your-org/skyguard-backend.git
    cd skyguard-backend
    ```
  </Accordion>
  <Accordion icon="download" title="Install Dependencies">
    ```bash
    npm install
    # or
    yarn install
    ```
  </Accordion>
</AccordionGroup>

### Step 2: Environment Configuration

<AccordionGroup>
  <Accordion icon="gear" title="Setup Environment Variables">
    1. Copy the example environment file:
    ```bash
    cp .env.example .env
    ```
    
    2. Update the `.env` file with your local configuration:
    ```bash
    NODE_ENV=development
    PORT=3001
    DATABASE_URL=postgresql://postgres:postgres@localhost:5432/skyguard_dev
    JWT_SECRET=your-development-secret-key-32-chars-min
    ```
    
    <Warning>
    Make sure to use a secure JWT secret in production environments.
    </Warning>
  </Accordion>
  <Accordion icon="database" title="Start Database Services">
    Use Docker Compose to start the required services:
    ```bash
    docker-compose up -d postgres
    ```
    
    This will start:
    - PostgreSQL database on port 5432
    - Adminer database management tool on port 8080
  </Accordion>
</AccordionGroup>

### Step 3: Database Setup

<AccordionGroup>
  <Accordion icon="play" title="Run Database Migrations">
    ```bash
    npm run db:migrate
    # or
    yarn db:migrate
    ```
  </Accordion>
  <Accordion icon="seedling" title="Seed Initial Data (Optional)">
    ```bash
    npm run db:seed
    # or
    yarn db:seed
    ```
  </Accordion>
</AccordionGroup>

### Step 4: Start Development Server

<Accordion icon="rocket" title="Launch the Application">
  ```bash
  npm run dev
  # or
  yarn dev
  ```
  
  Your SkyGuard backend will be available at:
  - **API Server**: `http://localhost:3001`
  - **API Documentation**: `http://localhost:3001/docs`
  - **Health Check**: `http://localhost:3001/api/v1/health`
  
  <Tip>
  The development server automatically restarts when you make changes to the code.
  </Tip>
</Accordion>

## Verify Your Setup

### Test the API

<CodeGroup>
```bash cURL
curl http://localhost:3001/api/v1/health
```

```javascript Fetch
fetch('http://localhost:3001/api/v1/health')
  .then(response => response.json())
  .then(data => console.log(data));
```

```bash HTTPie
http GET localhost:3001/api/v1/health
```
</CodeGroup>

You should receive a response like:
```json
{
  "success": true,
  "message": "SkyGuard API is healthy",
  "timestamp": "2025-09-22T10:30:00.000Z",
  "version": "1.0.0"
}
```

### Access API Documentation

Visit `http://localhost:3001/docs` to explore the interactive API documentation powered by Scalar.
